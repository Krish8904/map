# Interactive Map App

## Description

The Interactive Map App allows users to explore locations, search addresses, and add custom markers on a map. It provides a dynamic and responsive interface for map navigation.

## Features

* Search for locations using keywords or addresses.
* Zoom in/out and navigate the map.
* Add custom markers to specific locations.
* Responsive layout for desktop and mobile devices.
* Dynamic updating of the map based on user interaction.

## Technologies Used

* HTML
* CSS
* JavaScript
* Map API (e.g. OpenStreetMap)

## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in a web browser.
3. Use the search input to locate places.
4. Navigate the map using zoom and drag features.
5. Add markers by clicking on the desired locations.

## Validation

* Ensures that search input is not empty.
* Handles invalid location searches gracefully.
